# CorpAI Portal

![Portal Hero](../CorpAI/assets/hero.png)

> The premium Visual Operating System (VOS) for the [CorpAI open standard](https://github.com/Arigitshub/CorpAI).

## 🚀 Vision
The CorpAI Portal is designed to provide a high-fidelity, real-time command center for digital workforces. It visualizes hierarchies, traces reporting lines, and helps operators manage their agent ecosystems through a glassmorphic, immersive interface.

## 🔥 Features
- **Live Sync**: Automatically fetches and visualizes the latest CorpAI spec from the global registry.
- **Interactive Map**: Zoom, pan, and explore 50+ role hierarchies with `reactflow`.
- **Premium UI**: 8K-ready glassmorphic design system with smooth Framer Motion animations.
- **Agent Analytics**: Quick-view stats on department coverage and Rank distributions.

## 🛠️ Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS 4
- **Visualization**: React Flow
- **Animations**: Framer Motion
- **Icons**: Lucide React

## 🌎 Published to the World
This portal acts as the global registry viewer for the world's workers. Better than any static chart. Built for the agents.
